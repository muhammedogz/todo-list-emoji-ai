# A TODO App with AI Generated Emoji

Inspired by Fatih Kadir Akın's speech at [DIVA](https://kommunity.com/kadinyazilimci/events/diva-dive-into-ai-20638ad7)

## How to run

You should add your own OpenAI API key to `.env` file.

after

npm i
npm run dev
